2. Core Privacy Commitment: No Bank Link Aur Local Data Storage

BudgetMan app aapki privacy ko sabse upar rakhta hai. Humara app is tareeke se design kiya gaya hai ki aapka financial data sirf aapke control mein rahe.

  • NO BANK LINKING REQUIRED: BudgetMan aapse kabhi bhi aapke bank account details, login credentials, ya credit card information nahi maangta aur na hi link karta hai [Query text].
  • 100% LOCAL DATA STORAGE: Aapka poora financial data (expenses, income, budget, goals) sirf aapke device par hi locally store kiya jaata hai.[1] Hum aapka data kabhi bhi apne server par upload, sync, ya share nahi karte.

3. Data Collection Aur Iska Istemal (Usage and Purpose)

Chunki aap BudgetMan mein data manually enter karte hain, isliye humara data processing bahut limited hai. Hum sirf woh data dekhte hain jo aap app mein daalte hain:

Data Jo Hum Collect Karte Hain:

  • User-Inputted Financial Data: Income entries, expenses, transaction details, aur Custom Categories ke naam.
  • Financial Goals: Aapke set kiye gaye saving goals.
  • Technical Data: App crashes ko fix karne aur performance behtar karne ke liye sirf non-personal technical logs (crash reports, app usage) ka istemal karte hain.

Data Usage Purpose:

Aapka data sirf teen (3) internal zarooraton ke liye use hota hai:

  1. Core Functionality: Aapko reports, charts, aur analytics dikhaane ke liye.
  2. Smart AI Alerts & Suggestions: Aapke spending habits ko analyze karke personalized financial tips aur reminders dene ke liye.[2] Yeh analysis aapke device par hi hota hai.
  3. App Stability: Bugs aur technical issues ko theek karne ke liye.

4. Data Sharing Aur Third-Party Services

Hum aapka sensitive financial data kisi bhi third party ke saath marketing, advertising, ya anya maksad ke liye share nahi karte.[1, 3]

  • Financial Data Sharing: NONE. Aapka core financial data (expenses, income) aapke device ko chhodkar kahin nahi jaata.
  • Third-Party Tools: Hum sirf standard tools (jaise Google Crashlytics) ko app ki stability ke liye use kar sakte hain, lekin ismein aapka personal financial data shamil nahi hota.

5. Data Retention, Deletion, Aur Aapke Adhikaar (User Rights)

Aapke paas apne data par poora control hai. Humari Data Retention aur Deletion Policy yeh hai:

  • Data Retention Policy: Hum aapka data tab tak rakhte hain jab tak app aapke device par install rehta hai.
  • Data Deletion: Aap jab chahein apna poora data turant delete kar sakte hain. Kyunki data server par nahi hai, deletion instant hai. Aap iske liye app settings mein "Clear All Data" option use kar sakte hain ya simply app ko uninstall kar sakte hain.[1]

6. Policy Mein Badlaav (Updates)

Hum is policy ko badal sakte hain. Agar hum data collection practices mein koi bada badlaav laate hain (jaise ki agar hum server-side storage shuru karte hain), toh hum aapko Play Store listing ya app ke zariye turant notify karenge.[4]
